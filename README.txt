| Kode sumber program ada di folder src.                                                                                            |
= File executable dari program kalkulator ini ada di folder bin.                                                                    =
|                                                                                                                                   |
= Untuk melakukan kompilasi pada kode sumber dapat digunakan GNU Compiler Collection C, dengan menggunakan perintah                 =
| 	gcc main.c -o main.exe                                                                                                      |
= di folder kode sumber yaitu src. Setelah perintah dijalankan maka akan dibuat sebuah file executable main.exe yang merupakan      =
| hasil kompilasi dari kode sumber di main.c.                                                                                       |
=                                                                                                                                   =
| Program kalkulator dapat langsung digunakan dengan menjalankan file main.exe di folder bin. Selain itu, dapat program dapat       |
= dijalankan melalui terminal, command prompt, maupun powershell.                                                                   =
|                                                                                                                                   |
= Program akan menerima input dari pengguna, lalu mengevaluasi input tersebut. Jika input tersebut merupakan ekspresi matematika,   =
| maka program akan mengeluarkan hasil dari ekspresi tersebut. Jika input tersebut bukan merupakan ekspresi matematika, maka program|
= akan mengeluarkan output "Ekspresi tidak valid".                                                                                  =
|                                                                                                                                   |
= Untuk keluar dari program, ketik masukan "quit".                                                                                  =
|                                                                                                                                   |
